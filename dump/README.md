# Dump Programming Lang
Dump (Dont Use Me Please) , is made to practice making a [programming language](https://craftinginterpreters.com/scanning.html)
 


# Versions
# 0.1
the goal of 0.1 is to run a programe like this
```
    a <- 2;           # a = 2
    b <- 5 , c <- 69; # b = 5 , c = 5
    x <- a + b -> 5;  # x = a + b == 5
    whats x;         # output x
```
